移植Lua到ARM平台

发表于 2015-10-28   |   分类于 linux   |  

1.下载Lua源代码：

  • http://www.lua.org/versions.html#5.3

2.解压并进入目录lua-5.2.3/src

*

3.修改并保存Makefile文件（共两处）：

  • 第一处：第7行PLAT= none修改为PLAT= ansi
  • 第二处：第9行CC= gcc 修改为CC= arm-linux-gcc

4.执行make命令

5.很多.o库文件，一个lua可执行文件（解释器）和一个luac可执行文件（编译器，产生中间代码）。

  • 如果不打算在你的ARM开发板上使用那些库而只是想运行Lua脚本，则只需要将lua文件复制到开发板即可。写个简单的脚本试试吧：
  • –[测试脚本: test.lua–]
