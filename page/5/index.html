pycharm 设置git

发表于 2015-10-28   |   分类于 Python   |  

1.配置git

  • 打开file，选择settings，找到Version Contorl，打开
  • 找到GitHub ，HOST填github.com，用户名，密码，test，稍等一会，会提示成功
  • 设置好后选择git，这里是输入你的git.exe的,下面是我的git.exe的位置：
  • E:\Program Files (x86)\Git\bin\git.exe

2.clone代码

  • 设置好以后
  • 在主界面中打开VCS菜单
  • 找到checkout from version control
  • 点击GitHub
  • 打开以后就可以发现有三个输入框
  • 第一个是你在GitHub上创建的项目文件夹（我的是myflask）
  • 第二个是你想要放置myflask的位置（也就是myflask会下载到这个地方）
  • 第三个是文件夹名称
  • 点击Clone,用过GitHub的肯定知道，这个就是把你在GIT上创建的项目文件夹同步到本地
  • 稍等一会儿，你会发现你的PyCharm的项目栏中多了一个文件夹，我的是myflask

3.提交代码

  • a.选中项目名称–右键–git–commit…–
  • b.选中项目名称–右键–git–repository–push
